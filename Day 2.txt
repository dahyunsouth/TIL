* pull과 clone의 차이
새로운 환경에서 처음 다운로드 - clone
ㄴ git clone (url)
ㄴ 바탕화면 - git bash 열기 (* git init 지양, 찾기 어려움) - 경로 잘 설정돼있나 확인 - git clone (url)
push나 clone을 전에 진행한 후에 다운로드 - pull
ㄴ push나 clone 이력 있기 때문에 url 입력 불필요
ㄴ git pull origin +master(변경사항 없으면 pull 안 됨)
